Testing:
	- To test the software meeting all requirements, we use a Google Chrome browser. 
	Type...
		- http://localhost:3000
			- Default is Basil.
		- http://localhost:3000/
			- Default is Basil.
		- http://localhost:3000/recipes
			- Default is Basil.
		- http://localhost:3000/recipes.html
			- Default is Cake.
		- http://localhost:3000/index.html
			- Default is Cake.
		- http://localhost:3000/recipes?ingredient=Basil,Cumin
			- This will show ingredients that include Basil and Cumin all together.

Attention:
	- Type whatever ingredient you want in search bar.
	- When typing more than one ingredient in search bar, don't space between the comma.
